# 🌊 MoodFlow

**AI-powered mood tracking application** built with Next.js 14, React, TypeScript, and Tailwind CSS.

![MoodFlow](https://japanclassicstore-cyber.github.io/moodflow/images/hero-1.jpg)

## ✨ Features

- 🎭 **Mood Tracking** - Log your daily mood with 5 emoji levels (🤩 😊 😐 😔 😤)
- 📊 **Dashboard** - View mood statistics, trends, and insights
- 📝 **Journal** - Write notes and tag your entries
- 🌙 **Dark Mode** - Automatic light/dark theme switching
- 📱 **Responsive** - Works on all devices
- 🎨 **Beautiful UI** - Glassmorphism design with gradient backgrounds

## 🏃 Running Locally

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## 📝 Data Storage

All mood entries are stored in your browser's localStorage - your data stays private!
